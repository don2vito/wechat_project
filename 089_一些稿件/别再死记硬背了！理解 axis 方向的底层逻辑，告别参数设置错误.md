在使用 `numpy` 和 `pandas` 编写代码时，`axis` 参数该如何设置往往会傻傻分不清楚，这一问题相信也困扰了不少人，今天这篇文章就来解决这一痛点。

假设你有一个班级的成绩单（二维数组 / 表格）：

| 学生 | 数学 | 语文 | 英语 |
| :--- | :--- | :--- | :--- |
| 张三 | 90   | 85   | 88   |
| 李四 | 78   | 92   | 80   |
| 王五 | 85   | 88   | 90   |

```python
import numpy as np
import pandas as pd

# 创建一个成绩单表格（二维数组）
scores = np.array([[90, 85, 88],
                   [78, 92, 80],
                   [85, 88, 90]])

# 或者用pandas DataFrame
df = pd.DataFrame(scores, 
                  index=['张三', '李四', '王五'],
                  columns=['数学', '语文', '英语'])
print(df)
#      数学  语文  英语
# 张三   90   85   88
# 李四   78   92   80
# 王五   85   88   90
```

**轴是什么？就是方向！**

- **axis=0**：垂直方向 ↓（上下）
- **axis=1**：水平方向 →（左右）

就像数学里的坐标轴：

- **x轴**是水平的（→） 对应 **axis=1**
- **y轴**是垂直的（↓） 对应 **axis=0**

### 一、关键记忆法：**"沿着哪个方向移动"**

**1. axis=0（垂直方向，行方向）**

- **从上到下**读每一行的数据
- "沿着行方向" → 从第 0 行到第 1 行到第 2 行

```python
# 求每门课的平均分（对每个科目，计算所有学生的分数）
# 沿着axis=0（垂直方向）计算
average_by_subject = df.mean(axis=0)
print("每门课的平均分（axis=0）：")
print(average_by_subject)
# 数学    84.333333
# 语文    88.333333
# 英语    86.000000
# dtype: float64
```

**怎么想**：我要计算"数学"的平均分，就要沿着axis=0（垂直方向）把数据“压缩”。

**2. axis=1（水平方向，列方向）**

- **从左到右**读每一列的数据
- "沿着列方向" → 从第 0 列到第 1 列到第 2 列

```python
# 求每个学生的平均分（对每个学生，计算所有科目的分数）
# 沿着axis=1（水平方向）计算
average_by_student = df.mean(axis=1)
print("\n每个学生的平均分（axis=1）：")
print(average_by_student)
# 张三    87.666667
# 李四    83.333333
# 王五    87.666667
# dtype: float64
```

**怎么想**：我要计算"张三"的平均分，就要沿着axis=1（水平方向）把数据“压缩”。

### 二、轴编号的规律

#### 规律1：**从外到内编号**

多维数组可以看作"套娃"：

```python
# 创建一个三维数组：3个班级，每个班有上面的成绩单
three_classes = np.array([
    [[90, 85, 88],  # 班级1，学生1
     [78, 92, 80],  # 班级1，学生2
     [85, 88, 90]], # 班级1，学生3
    
    [[85, 90, 87],  # 班级2，学生1
     [80, 85, 82],  # 班级2，学生2
     [88, 92, 85]], # 班级2，学生3
    
    [[92, 88, 90],  # 班级3，学生1
     [85, 90, 87],  # 班级3，学生2
     [90, 85, 92]]  # 班级3，学生3
])

print(f"数组形状：{three_classes.shape}")  # (3, 3, 3)
```

**编号规律**：

- `axis=0`：最外层的括号（班级之间）
- `axis=1`：中间层的括号（学生之间）
- `axis=2`：最内层的括号（科目之间）

#### 规律2：**括号配对法**

数括号对，从外向内：

```python
# 三维数组的括号结构
[
  [ [90, 85, 88], [78, 92, 80], [85, 88, 90] ],  # ← axis=0：整个数组
  [ [85, 90, 87], [80, 85, 82], [88, 92, 85] ],  # ↑ 班级维度
  [ [92, 88, 90], [85, 90, 87], [90, 85, 92] ]
]
# 在axis=0内部：
[
  [90, 85, 88],  # ← axis=1：每个班级内部
  [78, 92, 80],  # ↑ 学生维度
  [85, 88, 90]
]
# 在axis=1内部：
[90, 85, 88]  # ← axis=2：每个学生内部
              # ↑ 科目维度
```

#### 实战练习：判断轴的方向

```python
# 练习1：二维数组求和
arr_2d = np.array([[1, 2, 3],
                   [4, 5, 6]])

print("原始数组：")
print(arr_2d)
print(f"形状：{arr_2d.shape}")  # (2, 3)

# 问题：下面这些操作分别沿着哪个轴？
print("\n1. 沿着axis=0求和（垂直方向，压缩行）：")
print(arr_2d.sum(axis=0))
# 结果：[5 7 9]
# 解释：1+4=5, 2+5=7, 3+6=9
# 把两行"压缩"成一行，列数不变

print("\n2. 沿着axis=1求和（水平方向，压缩列）：")
print(arr_2d.sum(axis=1))
# 结果：[6 15]
# 解释：第一行1+2+3=6, 第二行4+5+6=15
# 把三列"压缩"成一列，行数不变

# 练习2：删除操作
print("\n3. 沿着axis=0删除第一行（删除垂直方向的一行）：")
print(np.delete(arr_2d, 0, axis=0))
# 结果：[[4 5 6]]
# 解释：删除第0行，剩下第1行

print("\n4. 沿着axis=1删除第一列（删除水平方向的一列）：")
print(np.delete(arr_2d, 0, axis=1))
# 结果：[[2 3]
#       [5 6]]
# 解释：删除第0列，剩下第1、2列
```

### 三、快速判断诀窍

#### 诀窍1：**"压缩法"**

- 沿着`axis=0`操作 → 行数可能减少（压缩行）
- 沿着`axis=1`操作 → 列数可能减少（压缩列）

```python
def axis_cheat_sheet(data, axis):
    """快速判断axis效果的函数"""
    if axis == 0:
        return f"沿着axis={axis}（垂直方向）：压缩行，保留列特征"
    elif axis == 1:
        return f"沿着axis={axis}（水平方向）：压缩列，保留行特征"
    else:
        return f"沿着axis={axis}：操作更高维度"
```

#### 诀窍2：**"问自己问题"**

- 如果你想知道**每列**的统计量 → `axis=0`
  - "每门课的平均分是多少？"
  - "每列的最大值是多少？"
- 如果你想知道**每行**的统计量 → `axis=1`
  - "每个学生的总分是多少？"
  - "每行的最小值是多少？"

#### 诀窍3：**"看形状变化"**

```python
# 观察形状变化来判断axis
arr = np.random.rand(3, 4)  # 形状：(3, 4)

# 沿着axis=0求和 → 形状变成(4,)
# 因为3行被"压缩"成1行，只剩4列
result_axis0 = arr.sum(axis=0)
print(f"arr.sum(axis=0)的形状：{result_axis0.shape}")  # (4,)

# 沿着axis=1求和 → 形状变成(3,)
# 因为4列被"压缩"成1列，只剩3行
result_axis1 = arr.sum(axis=1)
print(f"arr.sum(axis=1)的形状：{result_axis1.shape}")  # (3,)
```

#### 高维数组的轴判断

对于三维及以上的数组，想象一个立方体：

```python
# 三维数组：形状(2, 3, 4)
# 可以想象成：2页纸，每页3行4列
cube = np.array([
    [[1, 2, 3, 4],   # 第0页，第0行
     [5, 6, 7, 8],   # 第0页，第1行
     [9, 10, 11, 12]], # 第0页，第2行
     
    [[13, 14, 15, 16], # 第1页，第0行
     [17, 18, 19, 20], # 第1页，第1行
     [21, 22, 23, 24]]  # 第1页，第2行
])

print(f"三维数组形状：{cube.shape}")  # (2, 3, 4)

# axis=0：跨页操作（垂直方向，最外层）
print(f"沿着axis=0求和的形状：{cube.sum(axis=0).shape}")  # (3, 4)
# 解释：2页合并成1页，每页内的3行4列结构不变

# axis=1：跨行操作（垂直方向，中间层）
print(f"沿着axis=1求和的形状：{cube.sum(axis=1).shape}")  # (2, 4)
# 解释：每页内的3行合并成1行，页数和列数不变

# axis=2：跨列操作（水平方向，最内层）
print(f"沿着axis=2求和的形状：{cube.sum(axis=2).shape}")  # (2, 3)
# 解释：每行的4列合并成1列，页数和行数不变
```

#### 终极记忆口诀

**口诀1：** "0竖1横"

- `axis=0` → 垂直方向（竖着看）
- `axis=1` → 水平方向（横着看）

**口诀2：** "0跨行，1跨列"

- `axis=0` → 跨越多行操作
- `axis=1` → 跨越多个列操作

**口诀3：** "括号数数法"
看最外层的括号是`axis=0`，往里一层是`axis=1`，再往里是`axis=2`...

### 四、实际应用场景

### 场景1：数据清洗（pandas）

```python
# 计算每列的缺失值数量（沿着axis=0）
import pandas as pd
import numpy as np

data = pd.DataFrame({
    '语文': [85, 92, np.nan, 88],
    '数学': [90, np.nan, 85, 92],
    '英语': [88, 85, 90, np.nan]
})

print("原始数据：")
print(data)

# 每列有多少缺失值？→ axis=0（垂直方向看每列）
missing_by_column = data.isnull().sum(axis=0)
print("\n每列的缺失值数量（axis=0）：")
print(missing_by_column)

# 每行有多少缺失值？→ axis=1（水平方向看每行）
missing_by_row = data.isnull().sum(axis=1)
print("\n每行的缺失值数量（axis=1）：")
print(missing_by_row)
```

### 场景2：图像处理（numpy）

```python
# 图像是三维数组：(高度, 宽度, 颜色通道)
image = np.random.randint(0, 256, (100, 100, 3), dtype=np.uint8)

print(f"图像形状：{image.shape}")  # (100, 100, 3)

# 计算每个像素点的亮度（平均RGB值）→ axis=2（最内层，颜色维度）
brightness = image.mean(axis=2)
print(f"亮度图形状：{brightness.shape}")  # (100, 100)

# 计算每列的亮度平均值 → axis=0（垂直方向）
column_brightness = brightness.mean(axis=0)
print(f"每列平均亮度形状：{column_brightness.shape}")  # (100,)
```

### 五、总结

记住这三个关键点：

1. **方向**：`axis=0`是垂直方向（↓），`axis=1`是水平方向（→）
2. **操作**：沿着哪个轴操作，就会"**压缩**"哪个维度
3. **结果**：操作后，被操作的那个维度会变小（通常变成1或消失）

最实用的判断方法是：

- 问自己："我想要对什么做操作？"
- 如果是"对每一列做操作" → `axis=0`
- 如果是"对每一行做操作" → `axis=1`

就像看书一样：

- 垂直翻页（看不同行）→ `axis=0`
- 水平阅读（看一行中的不同字）→ `axis=1`

刚开始可能会混淆，多练习几次就能形成直觉。记住：**轴就是方向，编号就是维度从外到内的顺序**。
