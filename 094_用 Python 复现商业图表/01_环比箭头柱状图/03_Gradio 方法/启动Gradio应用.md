# 环比箭头柱状图 - Gradio应用 - 快速启动指南

## 概述

本指南将帮助您快速启动和使用环比箭头柱状图的 Gradio 交互式 Web 应用。

## 系统要求

- Python 3.8 或更高版本
- 以下 Python 库（已自动安装）：
  - Gradio (≥6.0)
  - Matplotlib (≥3.0)
  - Pandas (≥1.0)
  - NumPy (≥1.20)

## 快速启动

### 1. 打开终端/命令提示符

在项目目录下打开终端或命令提示符：
```
d:\project\用 Python 复现商业图表\01_环比箭头柱状图
```

### 2. 运行应用

执行以下命令启动应用：

```bash
python app_gradio.py
```

### 3. 访问应用

应用启动后，终端会显示类似以下信息：

```
Running on local URL:  http://127.0.0.1:7860
```

在浏览器中打开显示的地址（通常是 `http://127.0.0.1:7860`）即可使用应用。

## 功能使用指南

### 数据导入

1. 点击"📁 上传数据文件"按钮
2. 选择您的数据文件（支持 .csv、.xlsx、.xls、.txt 格式）
3. 文件大小限制：不超过 10MB
4. 上传成功后会显示：
   - 文件基本信息（文件名、大小、行数、列数）
   - 数据预览（前 5 行）

### 数据格式要求

您的数据文件应符合以下格式：

| 年份 | 区域1 | 区域2 | 区域3 | ... |
|------|-------|-------|-------|-----|
| 2023 | 100   | 200   | 150   | ... |
| 2024 | 120   | 220   | 180   | ... |
| 2025 | 150   | 250   | 200   | ... |

**注意**：
- 第一列必须是年份/时间标签
- 从第二列开始是数据列
- 至少需要 2 行数据（2 年）才能计算环比增长率
- 数据值应为非负数

### 颜色配置

1. 数据导入成功后，会自动为每个数据列显示颜色选择器
2. 点击颜色选择器可以：
   - 从预设颜色中选择
   - 输入自定义颜色（HEX 格式，如 #FF0000）
3. 颜色变更后，图表会立即更新
4. 点击"🔄 重置为默认颜色"按钮可以恢复初始颜色

### 图表可视化

1. 数据导入并验证通过后，图表会自动生成
2. 图表功能：
   - 🖱️ **鼠标悬停**：显示详细数据值和环比百分比
   - 🔍 **缩放**：使用鼠标滚轮或拖拽进行缩放
   - ↔️ **平移**：拖拽图表进行平移
3. 图表包含：
   - 堆叠柱状图（各区域数据）
   - 数据标签（柱状图内部显示数值）
   - 总计标签（柱状图顶部显示总和）
   - 环比箭头和增长率百分比

### 图片导出

1. 在图表下方找到导出选项
2. 选择导出格式：
   - **PNG**：推荐，质量高，兼容性好
   - **JPG**：文件小，适合网络传输
   - **PDF**：矢量格式，适合打印
   - **SVG**：矢量格式，可编辑
3. 选择分辨率：
   - 150 DPI：屏幕显示
   - 300 DPI：打印（推荐）
   - 600 DPI：高质量打印
4. 点击"📥 下载图片"按钮
5. 文件会自动下载到您的浏览器下载文件夹
6. 文件名格式：`环比箭头图_YYYYMMDD_HHMMSS.格式`

## 常见问题解答

### Q: 应用无法启动怎么办？

**A**: 请检查：
1. Python 版本是否为 3.8 或更高
2. 所有依赖库是否已安装：
   ```bash
   pip install gradio matplotlib pandas numpy
   ```

### Q: 上传文件后没有反应？

**A**: 请确认：
1. 文件格式是否正确（.csv, .xlsx, .xls, .txt）
2. 文件大小是否不超过 10MB
3. 数据是否符合要求的格式

### Q: 图表中中文显示为方块？

**A**: 应用会自动尝试配置中文字体。如果仍有问题，请确保：
1. Windows 系统：确保安装了微软雅黑或黑体
2. macOS 系统：确保安装了华文黑体
3. Linux 系统：确保安装了文泉驿微米黑

### Q: 导出的图片不够清晰？

**A**: 请在导出时选择更高的分辨率：
- 300 DPI 适合大多数打印需求
- 600 DPI 适合高质量印刷

### Q: 如何停止应用？

**A**: 在运行应用的终端窗口中按 `Ctrl + C` 即可停止应用。

### Q: 可以在其他设备上访问吗？

**A**: 默认情况下，应用只能在本机访问。如需局域网访问，请查看 Gradio 文档配置共享选项。
