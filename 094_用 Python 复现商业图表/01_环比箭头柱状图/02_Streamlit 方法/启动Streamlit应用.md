# Streamlit应用 - 快速启动指南

## 🚀 快速开始

### 1. 安装Streamlit

如果还没有安装Streamlit，请运行：

```bash
pip install streamlit
```

### 2. 启动应用

在项目目录下运行：

```bash
cd "d:\project\用 Python 复现商业图表\01_环比箭头柱状图"
streamlit run app_streamlit.py
```

### 3. 访问应用

启动后，浏览器会自动打开应用页面。如果没有自动打开，请在浏览器中访问：
```
http://localhost:8501
```

## 📋 应用功能说明

### 1️⃣ 数据导入功能
- **支持格式**：CSV、Excel(.xlsx, .xls)、文本文件(.txt)
- **文件验证**：自动验证文件格式和数据完整性
- **信息显示**：上传后显示文件名、大小、数据量
- **数据预览**：可展开查看前10行数据

### 2️⃣ 数据可视化功能
- **自动生成**：基于导入数据自动生成环比箭头柱状图
- **环比箭头**：清晰的箭头指示数据增减方向
- **自定义标题**：可设置图表标题
- **颜色配置**：每个区域可独立选择颜色
- **丰富元素**：数据标签、总数值、增长率标签、图例

### 3️⃣ 图片导出功能
- **多格式支持**：PNG、JPG、PDF、SVG
- **高分辨率**：默认300 DPI，可选150/300/600 DPI
- **一键下载**：直接下载到本地
- **自动命名**：文件名包含时间戳

## 💡 使用步骤

### 步骤1：上传数据文件
1. 点击"上传数据表文件"按钮
2. 选择您的CSV、Excel或文本文件
3. 等待文件上传和验证
4. 查看数据预览

**数据格式要求**：
- 第一列：年份（如 2021, 2022, 2023）
- 后续列：各区域数据（数值型）
- 至少需要2年数据

### 步骤2：配置图表
1. 输入图表标题（可选）
2. 为每个区域选择颜色（可选）
3. 点击"🎨 生成图表"按钮

### 步骤3：导出图片
1. 选择分辨率（推荐300 DPI）
2. 选择导出格式（推荐PNG）
3. 点击"⬇️ 下载图片"按钮

## 🔧 故障排除

### 问题1：Streamlit未安装
**错误**：`'streamlit' 不是内部或外部命令`

**解决**：
```bash
pip install streamlit
```

### 问题2：端口被占用
**错误**：`Address already in use`

**解决**：
```bash
streamlit run app_streamlit.py --server.port 8502
```

### 问题3：中文字体显示问题
**症状**：中文显示为方框

**解决**：确保系统已安装中文字体（Windows自带微软雅黑）

### 问题4：依赖库缺失
**错误**：`ModuleNotFoundError`

**解决**：
```bash
pip install pandas numpy matplotlib openpyxl
```
