## 1. 产品概述

### 1.1 产品简介
环比箭头柱状图是一个专业的商业数据可视化工具，使用 Python 和 Matplotlib 生成高质量的带环比箭头的堆叠柱状图。该工具能够直观地展示不同年份各区域的数据，并在年份之间添加直角样式的环比增长率箭头，便于数据分析和商业报告使用。

### 1.2 核心特性
- ✅ 生成堆叠柱状图展示多区域数据
- ✅ 在柱状图上标注各区域的具体数值
- ✅ 在柱状图顶部标注每年的总数值
- ✅ 在年份之间绘制直角样式的环比增长率箭头
- ✅ 在箭头中央标注环比增长率百分比
- ✅ 支持自定义配置（图表尺寸、颜色、字体大小等）
- ✅ 完善的 DataProcessor 类，提供 CSV/Excel 读取、缺失值处理、异常值检测等高级功能
- ✅ 数据格式：使用带千分位分隔符的整数格式
- ✅ 箭头标签与箭头线条精确对齐

### 1.3 应用场景
- 商业年度报告
- 销售数据分析
- 区域业绩对比
- 财务数据可视化
- 市场趋势分析

## 3. 主要功能模块说明

### 3.1 StackedBarChartWithArrow 类
核心图表类，用于生成堆叠柱状图和环比箭头。

#### 主要参数
- `data`: 二维列表或数组，形状为 [年份数, 区域数]
- `years`: 年份标签列表
- `regions`: 区域名称列表
- `colors`: 每个区域的颜色列表
- `config`: 可选的配置字典，用于覆盖默认配置

#### 配置选项
```python
config = {
    'figsize': (10, 10),              # 图表尺寸（宽，高）
    'bar_width': 0.55,                  # 柱状图宽度
    'dpi': 150,                         # 输出图片分辨率
    'output_path': 'output.png',        # 输出图片路径
    'title_fontsize': 18,               # 标题字体大小
    'label_fontsize': 14,               # 图例标签字体大小
    'tick_fontsize': 13,                # 坐标轴刻度字体大小
    'data_label_fontsize': 13,           # 数据标签字体大小
    'percentage_fontsize': 12,           # 增长率标签字体大小
    'arrow_color': '#333333',           # 箭头颜色
    'percentage_background': '#2c3e50', # 增长率标签背景色
    'percentage_color': 'white'          # 增长率标签文字颜色
}
```

### 3.2 DataProcessor 类
数据处理类，提供高级数据处理功能。

#### 主要功能
- CSV/Excel文件读取
- 缺失值处理（5种策略）
- 异常值检测与清洗（IQR/Z-score方法）
- 数据格式转换
- 数据验证机制
- 统计计算功能

#### 使用示例
```python
processor = DataProcessor()
processor.read_file('data.csv')  # 或 data.xlsx
processor.handle_missing_values(strategy='fill_zero')
processor.detect_outliers(method='iqr')
```

### 3.3 核心算法原理

#### 3.3.1 堆叠柱状图绘制
- 使用 matplotlib 的 `bar()` 函数，通过 `bottom` 参数实现堆叠效果
- 从底部开始逐层绘制各区域的柱状图
- 每个区域的底部位置 = 之前所有区域高度之和

#### 3.3.2 环比增长率计算
- 环比增长率 = (本期值 - 上期值) / 上期值 × 100%
- 使用 numpy 数组运算高效计算增长率
- 支持多个年份连续计算

#### 3.3.3 直角箭头绘制
- 采用三段折线绘制直角箭头
- 从第一年柱状图顶部向上垂直上升到峰值
- 水平移动到第二年柱状图上方
- 垂直下降到第二年柱状图顶部
- 在终点位置添加箭头指示方向

#### 3.3.4 标签对齐
- 百分比标签垂直中心点与箭头水平线段的垂直中线完全重合
- 标签使用 `va='center'` 实现垂直居中对齐
- 在不同数据值和屏幕尺寸下保持位置一致性

## 5. 故障排除方法

### 5.1 常见问题及解决方案

#### 问题 1：中文显示为方框或乱码
**症状**：图表中的中文显示为方框或乱码

**解决方案**：
- 确保已正确调用 `set_chinese_font()` 函数
- 检查系统是否安装了中文字体
- Windows 系统：确保安装了微软雅黑或黑体
- macOS 系统：确保安装了华文黑体
- Linux 系统：确保安装了文泉驿微米黑

#### 问题 2：ModuleNotFoundError
**症状**：运行时提示找不到模块
```
ModuleNotFoundError: No module named 'matplotlib'
```

**解决方案**：
```bash
pip install matplotlib numpy pandas openpyxl
```

#### 问题 3：数据格式错误
**症状**：提示数据维度不匹配
```
ValueError: 数据行数必须与年份数量匹配
```

**解决方案**：
- 检查 `data` 的第一维长度是否等于 `years` 的长度
- 检查 `data` 的第二维长度是否等于 `regions` 的长度
- 确保颜色列表长度等于区域数量

#### 问题 4：图表生成失败
**症状**：程序运行但没有生成图片文件

**解决方案**：
- 检查 `output_path` 路径是否有写入权限
- 确保输出目录存在
- 检查磁盘空间是否充足
- 查看错误信息，按照提示修复

#### 问题 5：百分比标签位置不对
**症状**：标签没有与箭头线条对齐

**解决方案**：
- 确保使用了最新版本的代码
- 检查 `label_y` 是否设置为 `peak_y`
- 确认使用了 `va='center'` 垂直居中对齐

#### 问题 6：Excel 文件读取失败
**症状**：无法读取 .xlsx 文件

**解决方案**：
```bash
pip install openpyxl
```

## 6. 注意事项

### 6.1 数据格式要求

#### 数据结构
- 数据必须是二维结构，第一维是年份，第二维是区域
- 至少需要 2 年的数据才能计算环比增长率
- 数据值应为非负数

#### 颜色配置要求
- 颜色数量必须与区域数量一致
- 建议使用对比度较高的颜色组合以提高可读性
- 推荐使用十六进制颜色代码（如 `#c82423`）

### 6.2 字体要求
- Windows 系统默认使用微软雅黑或黑体
- macOS 系统默认使用黑体
- Linux 系统默认使用文泉驿微米黑
- 如需其他字体请修改 `set_chinese_font()` 函数

### 6.3 输出文件
- 输出路径会自动创建父目录
- 支持多种图片格式（PNG、JPG 等）
- 建议使用 PNG 格式以保证最佳显示效果

### 6.4 性能建议
- 对于大量数据建议适当调整 `figsize` 和 `dpi`
- 调整 `bar_width` 以适应年份数量
- 垂直偏移量会根据数据最大值自动调整

### 6.5 数据格式说明
- 所有数值使用带千分位分隔符的整数格式（如 1,000）
- 已移除货币符号 $，提高通用性
- 不保留任何小数位，保持整洁

### 6.6 备份建议
- 在修改代码前备份原始文件
- 定期备份重要的数据文件
- 保存生成的图表到安全位置

### 6.7 更新说明
代码可能会不定期更新，建议关注：
- 字体大小调整
- 数据导入方式优化
