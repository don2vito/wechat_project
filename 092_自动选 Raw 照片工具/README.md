# RAW照片挑选工具 (HTML网页版)

一个基于HTML5的RAW照片挑选工具，可以直接在浏览器中运行，无需安装任何软件。

## 功能特点

- **纯前端实现**：无需服务器，直接在浏览器中运行
- **文件夹选择**：使用File System Access API选择源文件夹
- **照片预览**：自动扫描并显示JPG/JPEG照片缩略图
- **多种排序**：支持按文件名、修改日期、文件大小排序
- **全屏查看**：点击缩略图进入全屏查看模式
- **图片操作**：支持放大、缩小、旋转功能
- **键盘快捷键**：支持方向键快速浏览和选择
- **RAW导出**：自动匹配并导出同名RAW格式文件
- **进度指示**：显示文件复制进度
- **科技感UI**：深色主题，简洁美观

## 浏览器要求

本工具使用File System Access API，需要以下浏览器版本：

- Chrome 86+
- Edge 86+
- Firefox 90+
- Opera 72+

**注意**：由于安全限制，必须通过`http://localhost`或`https://`协议访问，直接打开文件可能无法使用文件夹选择功能。

## 使用方法

### 1. 启动工具

#### 方式一：使用本地服务器（推荐）

```bash
# 使用Python
python -m http.server 8000

# 使用Node.js
npx serve

# 使用PHP
php -S localhost:8000
```

然后在浏览器中访问 `http://localhost:8000`

#### 方式二：直接打开

双击 `index.html` 文件在浏览器中打开（部分功能可能受限）。

### 2. 选择来源文件夹

1. 点击"来源文件夹"按钮
2. 在弹出的对话框中选择包含照片的文件夹
3. 系统将自动扫描文件夹内的所有JPG/JPEG照片

### 3. 浏览和选择照片

- **缩略图视图**：在主界面查看所有照片的缩略图
- **排序**：使用下拉菜单按文件名、日期或大小排序
- **选择**：点击缩略图进入全屏查看模式
- **勾选**：在全屏模式下点击星形按钮或使用键盘快捷键

### 4. 全屏查看模式

在全屏查看模式下，你可以：

- **放大/缩小**：点击 +/- 按钮
- **旋转**：点击 ↺/↻ 按钮进行90度旋转
- **导航**：点击左右箭头或按键盘方向键
- **选择**：点击星形按钮或按上箭头键
- **退出**：按ESC键或点击✕按钮

### 5. 导出RAW文件

1. 点击"完成"按钮进入导出界面
2. 点击"选择导出文件夹"按钮选择目标文件夹
3. 系统将自动查找与所选JPG同名的RAW文件并复制到目标文件夹
4. 支持的RAW格式：CR2、NEF、ARW、3FR、DNG、DATA、SR2

## 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| ← (左箭头) | 上一张照片 |
| → (右箭头) | 下一张照片 |
| ↑ (上箭头) | 选择当前照片 |
| ↓ (下箭头) | 取消选择当前照片 |
| ESC | 关闭查看器 |
| F11 | 切换全屏模式 |

## 注意事项

1. **安全性**：由于浏览器安全限制，首次使用文件夹功能时需要用户授权
2. **性能**：处理大量照片时可能需要一些时间生成缩略图
3. **文件格式**：确保JPG和RAW文件在同一文件夹或子文件夹中
4. **文件名**：RAW文件需要与JPG文件同名（不区分大小写）

## 常见问题

### Q: 为什么无法选择文件夹？
A: 请确保使用支持的浏览器，并通过本地服务器访问（http://localhost）。

### Q: 为什么找不到RAW文件？
A: 请确保RAW文件与JPG文件同名，并且位于源文件夹或其子文件夹中。

### Q: 导出时提示失败？
A: 请检查目标文件夹是否有写入权限，以及RAW文件是否被其他程序占用。
