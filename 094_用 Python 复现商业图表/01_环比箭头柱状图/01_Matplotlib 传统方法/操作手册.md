## 1. 项目概述

### 1.1 产品简介
环比箭头柱状图是一个专业的商业数据可视化工具，使用 Python 和 Matplotlib 生成高质量的带环比箭头的堆叠柱状图。该工具能够直观地展示不同年份各区域的数据，并在年份之间添加直角样式的环比增长率箭头，便于数据分析和商业报告使用。

### 1.2 核心功能
- ✅ 生成堆叠柱状图展示多区域数据
- ✅ 在柱状图上标注各区域的具体数值
- ✅ 在柱状图顶部标注每年的总数值
- ✅ 在年份之间绘制直角样式的环比增长率箭头
- ✅ 在箭头中央标注环比增长率百分比
- ✅ 支持自定义配置（图表尺寸、颜色、字体大小等）
- ✅ 完善的 DataProcessor 类，提供 CSV/Excel 读取、缺失值处理、异常值检测等高级功能
- ✅ 数据格式使用带千分位分隔符的整数格式
- ✅ 箭头标签与箭头线条精确对齐

### 1.3 应用场景
- 商业年度报告
- 销售数据分析
- 区域业绩对比
- 财务数据可视化
- 市场趋势分析

## 4. 图片生成功能详细操作步骤

### 4.1 方式一：使用 Python 列表直接生成（快速上手）

#### 步骤 1：准备数据
在代码中准备您的数据：

```python
# 年份标签
years = ['2024', '2025', '2026']

# 区域名称
regions = ['东南亚', '北欧', '美国']

# 各区域颜色（颜色数量必须与区域数量一致）
colors = ['#c82423', '#e97451', '#f8b195']

# 数据：[年份数, 区域数]
data = [
    [48, 78, 80],   # 2024年
    [56, 120, 157], # 2025年
    [61, 202, 260]  # 2026年
]
```

#### 步骤 2：配置图表参数（可选）
```python
config = {
    'figsize': (10, 10),              # 图表尺寸（宽，高）
    'bar_width': 0.55,                  # 柱状图宽度
    'dpi': 150,                         # 输出图片分辨率
    'output_path': '我的图表.png',      # 输出图片路径
    'title_fontsize': 18,               # 标题字体大小
    'label_fontsize': 14,               # 图例标签字体大小
    'tick_fontsize': 13,                # 坐标轴刻度字体大小
    'data_label_fontsize': 13,          # 数据标签字体大小
    'percentage_fontsize': 12,          # 增长率标签字体大小
    'arrow_color': '#333333',           # 箭头颜色
    'percentage_background': '#2c3e50', # 增长率标签背景色
    'percentage_color': 'white'          # 增长率标签文字颜色
}
```

#### 步骤 3：创建图表并生成
```python
from 环比箭头柱状图 import StackedBarChartWithArrow, set_chinese_font

# 设置中文字体（必需）
set_chinese_font()

# 创建图表实例
chart = StackedBarChartWithArrow(data, years, regions, colors, config)

# 绘制并保存图表
chart.plot()

print("图表已成功生成！")
```

#### 步骤 4：查看结果
生成的图表将保存在配置的 `output_path` 路径中。

### 4.2 方式二：从 CSV/Excel 文件读取（推荐）

#### 步骤 1：准备数据文件
按照第 3 节的格式规范准备您的数据文件，例如 `mydata.csv`。

#### 步骤 2：使用 DataProcessor 读取数据
```python
from 环比箭头柱状图 import DataProcessor, StackedBarChartWithArrow, set_chinese_font

# 设置中文字体（必需）
set_chinese_font()

# 创建数据处理器
processor = DataProcessor()

# 读取数据文件
processor.read_file('mydata.csv')  # 或 'mydata.xlsx'

# 可选：处理缺失值
processor.handle_missing_values(strategy='fill_zero')

# 可选：检测异常值
outliers = processor.detect_outliers(method='iqr')
print(f"检测到的异常值: {outliers}")
```

#### 步骤 3：生成图表
```python
# 配置参数
config = {
    'figsize': (10, 10),
    'output_path': '从文件生成的图表.png',
    # ... 其他配置
}

# 区域颜色（颜色数量必须与区域数量一致）
colors = ['#3498db', '#2ecc71', '#e74c3c', '#f39c12', '#9b59b6']

# 从数据处理器创建图表
chart = StackedBarChartWithArrow(
    colors=colors,
    config=config,
    data_processor=processor
)

# 绘制图表
chart.plot()

print("图表已成功生成！")
```

### 4.5 DataProcessor 高级功能

#### 4.5.1 缺失值处理策略
| 策略 | 说明 | 适用场景 |
|------|------|----------|
| 'fill_zero' | 用0填充 | 缺失值表示"无"的情况 |
| 'fill_mean' | 用平均值填充 | 正态分布的数值型数据 |
| 'fill_median' | 用中位数填充 | 有异常值的数据 |
| 'fill_value' | 用指定值填充 | 需要自定义填充值 |
| 'drop' | 删除包含缺失值的行/列 | 缺失值较少且可接受数据减少 |

**使用示例：**
```python
processor.handle_missing_values(strategy='fill_zero')
```

#### 4.5.2 异常值检测方法
| 方法 | 说明 | 适用场景 |
|------|------|----------|
| 'iqr' | 四分位距方法 | 数据分布不对称或有明显异常值 |
| 'zscore' | Z-score方法 | 数据近似服从正态分布 |

**使用示例：**
```python
outliers = processor.detect_outliers(method='iqr')
print(outliers)
```

## 5. 常见问题排查与解决方案

### 5.1 安装问题

#### 问题 1：ModuleNotFoundError
**症状**：运行时提示找不到模块
```
ModuleNotFoundError: No module named 'matplotlib'
```

**解决方案**：
```bash
pip install matplotlib numpy pandas openpyxl
```

#### 问题 2：pip 命令不可用
**症状**：提示 'pip' 不是内部或外部命令

**解决方案**：
- 确保 Python 已正确添加到系统 PATH
- 或使用 `python -m pip` 代替 `pip`
```bash
python -m pip install matplotlib numpy pandas openpyxl
```

---

### 5.2 字体显示问题

#### 问题 3：中文显示为方框或乱码
**症状**：图表中的中文显示为方框或乱码

**解决方案**：
- 确保已正确调用 `set_chinese_font()` 函数
- 检查系统是否安装了中文字体
  - Windows 系统：确保安装了微软雅黑或黑体
  - macOS 系统：确保安装了华文黑体
  - Linux 系统：确保安装了文泉驿微米黑

---

### 5.3 数据格式问题

#### 问题 4：数据维度不匹配
**症状**：提示数据行数或列数不匹配
```
ValueError: 数据行数必须与年份数量匹配
```

**解决方案**：
- 检查 `data` 的第一维长度是否等于 `years` 的长度
- 检查 `data` 的第二维长度是否等于 `regions` 的长度
- 确保颜色列表长度等于区域数量

#### 问题 5：文件读取失败
**症状**：无法读取 CSV 或 Excel 文件

**解决方案**：
- 检查文件路径是否正确
- 确保文件存在
- 检查文件格式是否正确
- Excel 文件需要安装 openpyxl：
```bash
pip install openpyxl
```

---

### 5.4 图表生成问题

#### 问题 6：图表生成失败，没有生成图片
**症状**：程序运行但没有生成图片文件

**解决方案**：
- 检查 `output_path` 路径是否有写入权限
- 确保输出目录存在
- 检查磁盘空间是否充足
- 查看错误信息，按照提示修复

#### 问题 7：至少需要2年数据
**症状**：提示至少需要2年数据才能计算环比增长率

**解决方案**：
- 确保数据中至少包含2个年份
- 环比增长率需要至少两年数据才能计算

---

### 5.5 其他问题

#### 问题 8：柱状图重叠或显示不全
**症状**：柱状图重叠在一起或部分被裁剪

**解决方案**：
- 调整 `figsize` 增大图表尺寸
- 调整 `bar_width` 减小柱状图宽度
- 调整 `dpi` 提高分辨率

#### 问题 9：颜色显示不正确
**症状**：区域颜色与预期不符

**解决方案**：
- 确保颜色列表长度与区域数量一致
- 使用有效的颜色格式（十六进制代码或颜色名称）

### 6.5 输出图片内容说明

生成的图片包含以下元素：

1. **堆叠柱状图**
   - 每个年份对应一个柱状图
   - 每个柱状图按区域堆叠
   - 每个区域使用不同颜色

2. **数据标签**
   - 每个区域的数值显示在柱状图内部
   - 使用白色字体
   - 带千分位分隔符（如 1,000）

3. **总数值标签**
   - 每年的总数值显示在柱状图顶部
   - 使用加粗字体
   - 带千分位分隔符

4. **环比箭头**
   - 直角样式箭头连接相邻年份
   - 箭头方向指示增长或下降
   - 向下箭头表示负增长，向上箭头表示正增长

5. **增长率标签**
   - 圆形背景标签
   - 显示环比增长率百分比
   - 负增长直接显示，正增长前加"+"号

6. **图例**
   - 显示各区域名称和对应颜色
   - 位于图表顶部居中
   - 无边框

7. **坐标轴**
   - X轴：显示年份标签
   - Y轴：隐藏刻度（简化显示）
   - 隐藏上、右、左边框
