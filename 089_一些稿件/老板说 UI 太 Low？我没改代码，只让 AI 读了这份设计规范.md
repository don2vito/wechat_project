> https://github.com/anthropics/skills/blob/main/skills/frontend-design/SKILL.md

```plaintext
名称：前端设计
描述：创建独具特色、可用于生产环境的高品质前端界面。当用户要求构建网页组件、页面、设计成果、海报或应用程序时使用此技能（示例包括网站、落地页、数据看板、React 组件、HTML/CSS 布局，或对任意网页 UI 进行样式美化）。生成富有创意、制作精良的代码与 UI 设计，避免千篇一律的 AI 风格。
许可协议：完整条款详见 LICENSE.txt

---
本技能用于指导创建独具特色、可投入生产的前端界面，避免俗套的 “AI 劣质设计” 风格。编写可实际运行的代码，同时极致注重美学细节与创意设计。
用户会提供前端需求：需要搭建的组件、页面、应用程序或界面。可能会附带关于用途、目标用户或技术限制的相关说明。
设计思路
编码前，先理清使用场景，并确定鲜明的美学风格方向：
- 用途：该界面解决什么问题？用户群体是谁？
- 风格基调：选择一种极致风格：极简克制风、极繁杂乱风、复古未来风、自然有机风、奢华精致风、趣味童趣风、杂志排版风、粗野原始风、装饰艺术几何风、柔和浅色系风、工业实用风等。风格种类丰富，可从中汲取灵感，但最终设计需贴合选定的美学方向。
- 限制条件：技术要求（框架、性能、无障碍适配）。
- 差异化亮点：是什么让设计令人过目不忘？最让人印象深刻的核心亮点是什么？
重要提示：选定清晰的设计理念方向，并精准落地执行。大胆的极繁设计与精致的极简设计均可适用 —— 关键在于设计的目的性，而非风格的繁复程度。
随后编写可运行的代码（HTML/CSS/JS、React、Vue 等），需满足：
- 可用于生产环境，功能完整可用
- 视觉效果出众，令人印象深刻
- 风格统一，美学定位清晰
- 各处细节打磨精细
前端美学设计准则
重点关注：
- 排版：选用美观、独特、有辨识度的字体。避免 Arial、Inter 等通用字体，优先选择能提升前端美学质感的特色字体，采用新颖且有个性的字体搭配。将一款特色标题字体与一款精致正文字体组合使用。
- 色彩与主题：坚持统一的美学风格，使用 CSS 变量保证风格一致。主色调搭配鲜明的强调色，远优于平淡分散的配色方案。
- 动效：运用动画实现视觉效果与微交互。HTML 优先使用纯 CSS 动效方案，React 可搭配 Motion 动画库。聚焦高冲击力的动效节点：一个编排精良、分步依次呈现的页面加载动效（通过动画延迟实现），比零散的微交互更具愉悦感。添加滚动触发动效与出人意料的悬停状态效果。
- 空间布局：采用非常规布局、不对称设计、元素叠加、斜向视觉动线、突破网格的元素设计。保留充足留白，或控制合理的元素密度。
- 背景与视觉细节：营造氛围与视觉层次感，而非单纯使用纯色背景。添加贴合整体风格的场景化效果与纹理，运用渐变网格、杂色纹理、几何图案、多层透明效果、强烈阴影、装饰边框、自定义光标、颗粒质感叠加等创意形式。
严禁使用通用的 AI 生成式设计，例如滥用 Inter、Roboto、Arial 及系统默认字体，俗套配色方案（尤其是白底紫色渐变），刻板的布局与组件样式，以及缺乏场景特色的模板化设计。
进行创意解读，做出贴合场景、出人意料的设计选择。杜绝千篇一律的设计，在明暗主题、字体选用、风格方向上灵活变化。切勿重复使用常见字体（例如 Space Grotesk）。
重要提示：代码实现复杂度需匹配美学设计构想。极繁风格设计需搭配复杂代码，实现丰富的动画与特效；极简或精致风格设计则需克制、精准，着重把控间距、排版与细腻细节。优雅的设计源于对设计构想的完美落地。
谨记：本工具具备出色的创意创作能力。大胆发挥，跳出思维定式，全力践行独特的设计理念，展现真正的创作潜力。
```
