# DataQuery Pro - 数据查询分析系统

一个基于 Python + DuckDB 的专业数据查询分析平台，提供美观的界面和强大的数据处理能力。

## 功能特性

### 核心功能
- **数据导入**：支持 Excel (.xlsx, .xls)、CSV、TXT 等多种格式
- **SQL 查询**：支持标准 SQL 语法，具备语法高亮和自动补全
- **数据可视化**：支持柱状图、折线图、饼图等多种图表类型
- **结果导出**：查询结果可导出为 Excel 格式

### 高级功能
- **字段配置**：自动识别数据类型，支持手动调整
- **数据备份**：一键备份和恢复数据
- **用户认证**：安全的登录注册机制
- **树状导航**：直观展示数据表和字段结构

## 使用指南

### 1. 注册与登录
- 首次使用需注册账号
- 输入用户名和密码完成注册
- 使用注册的账号登录系统

### 2. 数据导入
1. 点击顶部导航栏的"导入"
2. 拖放文件到上传区域或点击选择文件
3. 系统自动识别字段类型
4. 可手动调整字段数据类型
5. 点击"确认导入"完成数据导入

### 3. 数据查询
1. 左侧树状图显示已导入的数据表
2. 点击表名展开查看字段
3. 点击字段名可快速插入到 SQL 编辑器
4. 在编辑器输入 SQL 查询语句
5. 点击"执行查询"或按 Ctrl+Enter 执行
6. 下方显示查询结果

### 4. 数据可视化
1. 点击顶部导航栏的"可视化"
2. 选择数据表和字段
3. 选择图表类型（柱状图/折线图/饼图等）
4. 配置聚合方式
5. 点击"生成图表"

### 5. 数据导出
1. 执行查询后，点击"导出"按钮
2. 自动生成 Excel 文件并下载
3. 导出文件保留原始数据类型

### 6. 数据备份
1. 点击顶部导航栏的"备份"
2. 点击"创建备份"按钮
3. 备份列表显示所有历史备份
4. 可随时恢复到任意备份点

## SQL 示例

```sql
-- 查询所有数据
SELECT * FROM 表名 LIMIT 100;

-- 条件查询
SELECT * FROM 表名 WHERE 字段 > 100;

-- 聚合查询
SELECT 类别, COUNT(*) as 数量, SUM(金额) as 总金额
FROM 表名
GROUP BY 类别;

-- 排序
SELECT * FROM 表名 ORDER BY 字段 DESC;

-- 多表关联
SELECT a.*, b.名称
FROM 表A a
JOIN 表B b ON a.id = b.id;
```

## 常见问题

### Q: 启动失败怎么办？
A: 请检查 Python 版本是否为 3.8+，并确保所有依赖已安装。

### Q: 如何处理大数据量？
A: DuckDB 专为分析查询优化，支持百万级数据的快速查询。

### Q: 支持哪些文件格式？
A: 支持 Excel (.xlsx, .xls)、CSV、TXT（自动检测分隔符）。

### Q: 如何进行数据备份？
A: 在"备份"页面点击"创建备份"，系统会自动备份所有数据。
