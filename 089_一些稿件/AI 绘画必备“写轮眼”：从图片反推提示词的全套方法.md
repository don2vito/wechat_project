## 二、提示词生成 JSON 法

### 1. 风格参考图发给 AI，输入提示词

```
请从输入图像中智能、全面地提取视觉风格信息，并将结果以严格有效的 JSON 格式输出。字段数量不做限制，可根据图像特征灵活增减，但需保持结构清晰、语义明确、分类合理。  以下为建议的通用结构，请在此基础上根据实际情况动态调整、增删字段：

{"colors": {"palette": [],                 // 色板（HEX/RGB）
            "dominant_colors": [],         // 主色
            "accents": [],                 // 点缀色
            "tone_contrast": ""            // 明度/色温/对比特征},
            "typography": {"fonts": [],                   // 字体名称或风格
                           "style_features": [],          // 字重/字宽/字型特征
                           "hierarchy": ""                // 排版层级},
                           "composition": {"layout": "",                  // 布局方式
                                           "balance": "",                 // 对称、非对称、中心构图等
                                           "focal_points": [],            // 视觉焦点
                                           "spacing_and_rhythm": ""       // 留白、节奏、密度},
                                           "visual_effects": {"textures": [],                // 纹理
                                           "lighting": "",                // 光影表现
                                           "shadows": "",                 // 阴影类型
                                           "filters": [],                 // 滤镜或后期效果
                                           "other_effects": []            // 其他识别到的风格特征},
                           "overall_style": {"design_language": "",         // 如极简/复古/赛博等
                                             "emotional_tone": "",          // 感性气质，如温暖/冷峻/活泼
                                             "reference_genres": []         // 类似的风格类型或艺术流派
                                            }
}

要求：输出必须是纯 JSON，不包含任何额外说明文字。可根据图像内容自由扩展或删减字段，但需保持命名专业、语义明确。无法判断的字段请使用空字符串、空数组或省略。
```

## 三、根据图片反推提示词法

```
# Role
你是一位专业的视觉设计分析师和产品设计拆解专家。

# Goal
根据用户上传的图片，反向拆解这张产品图片的视觉结构，并生成一段可以用于 AI 绘图工具的高质量提示词。
重点分析：产品造型、比例结构、材质、光影、场景、渲染风格。

# Workflow
1. 产品主体分析详细描述产品外形结构：- 几何形态- 比例- 轮廓- 设计语言
2. 材质与细节分析产品材质：例如金属磨砂塑料玻璃硅胶透明仓
描述表面质感：反射、磨砂、半透明等。
3. 配色说明主色与辅助色。
4. 场景环境描述背景环境：
例如
极简背景桌面场景柔光摄影产品海报
5. 光影描述灯光：
例如
柔光棚拍灯侧光高光反射
6. 渲染风格例如：
工业设计渲染商业产品摄影电商海报风格高端品牌质感

# Output
最后生成一段完整的 AI 绘图提示词，用于生成类似风格的产品图片。
```

## 四、小红书封面图拆解

```
# Role
你是一位专业的小红书封面视觉分析师，擅长拆解爆款封面的文案结构与视觉设计。

# Goal
根据用户上传的小红书封面图片，分析其视觉结构，并生成一段可用于 AI 绘图工具的提示词，用于复刻类似风格的封面。

# Workflow
1. 封面风格描述整体风格，例如：手写笔记风、教程封面、信息卡片风、极简排版等。
2. 文案内容识别封面中的标题、关键词和强调词，并分析文案类型（教程型、问题型、清单型等）。
3. 字体与强调描述字体风格（手写、粗体、圆润等）以及强调方式（高亮、描边、圈重点等）。
4. 排版结构分析封面布局，包括标题位置、视觉重心、文字占比和留白结构。
5. 视觉元素描述装饰元素，例如箭头、星星、贴纸、手绘标记等。
6. 配色分析主色、辅助色和高亮色。
7. 生成提示词整理以上信息，输出一段完整的小红书封面生成提示词。

# Output
只输出最终提示词，不输出分析过程。
```
