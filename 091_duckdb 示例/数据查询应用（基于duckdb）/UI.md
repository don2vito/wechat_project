# DataQuery Pro UI 设计方案

## 1. 设计概述

### 1.1 设计理念
DataQuery Pro 采用现代深色主题设计，以"专业、高效、优雅"为核心设计理念。通过精心设计的视觉层次和交互细节，为用户提供舒适的数据分析体验。

### 1.2 设计风格
- **主题**：深色模式 (Dark Mode)
- **色彩**：以靛蓝 (#6366f1) 为主色调，搭配青色 (#0ea5e9) 和琥珀色 (#f59e0b) 作为辅助
- **字体**：思源黑体 (Noto Sans SC) + JetBrains Mono 等宽字体
- **圆角**：6px-16px 渐进式圆角
- **阴影**：多层次阴影营造空间感

## 8. 交互设计

### 8.1 动画效果

#### 页面切换
```css
transition: opacity 0.3s ease, transform 0.3s ease;
```

#### 按钮悬停
```css
transform: translateY(-2px);
box-shadow: 0 8px 24px rgba(99, 102, 241, 0.5);
```

#### Toast 提示
```css
@keyframes slideIn {
    from { transform: translateX(100%); opacity: 0; }
    to { transform: translateX(0); opacity: 1; }
}
```

#### 加载动画
```css
@keyframes spin {
    to { transform: rotate(360deg); }
}
```

### 8.2 交互反馈

| 操作 | 反馈 |
|------|------|
| 点击按钮 | 按下动画 + 颜色变化 |
| 悬停可点击元素 | 背景色变化 + 手型光标 |
| 输入框聚焦 | 边框高亮 + 阴影 |
| 加载中 | 旋转动画 + 文字提示 |
| 操作成功 | 绿色 Toast 提示 |
| 操作失败 | 红色 Toast 提示 |

### 8.3 快捷键

| Ctrl+Enter | 执行 SQL 查询 |
