# DataQuery Pro 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 产品名称
DataQuery Pro - 数据查询分析系统

### 1.2 产品定位
DataQuery Pro 是一款面向数据分析师、业务人员和开发者的轻量级数据查询分析工具。基于 DuckDB 嵌入式分析型数据库，提供直观的 Web 界面，支持多数据源导入、SQL 查询、数据可视化和结果导出等功能。

### 1.3 目标用户
- 数据分析师：需要快速查询和分析数据
- 业务人员：需要查看数据报表和生成图表
- 开发者：需要调试 SQL 查询和验证数据
- 数据科学家：需要探索性数据分析

### 1.4 产品价值
- 降低数据分析门槛，无需安装复杂数据库软件
- 提供直观的可视化界面，提升数据分析效率
- 支持多种数据格式导入，灵活处理各类数据源
- 基于 DuckDB 的高性能查询引擎，快速响应分析需求

## 3. 功能需求

### 3.1 用户认证模块

#### 3.1.1 用户注册
- **功能描述**：新用户创建账号
- **输入项**：
  - 用户名：3-20 位字符，支持中英文、数字、下划线
  - 密码：6-20 位字符，至少包含字母和数字
  - 确认密码：与密码一致
- **业务规则**：
  - 用户名不可重复
  - 密码使用 SHA-256 加密存储
- **输出**：注册成功提示，自动跳转登录页

#### 3.1.2 用户登录
- **功能描述**：已注册用户登录系统
- **输入项**：
  - 用户名
  - 密码
- **业务规则**：
  - 验证用户名密码匹配
  - 登录成功后生成 Token
  - Token 有效期 24 小时
- **输出**：登录成功，进入主界面

#### 3.1.3 用户登出
- **功能描述**：退出当前登录状态
- **业务规则**：
  - 清除本地 Token
  - 跳转至登录页

---

### 3.2 数据导入模块

#### 3.2.1 文件上传
- **功能描述**：上传数据文件到系统
- **支持格式**：
  - Excel: .xlsx, .xls
  - CSV: .csv
  - TXT: .txt（自动检测分隔符）
- **上传方式**：
  - 点击选择文件
  - 拖放文件到上传区域
- **业务规则**：
  - 文件大小限制：100MB
  - 显示上传进度条
  - 自动检测文件编码（UTF-8, GBK 等）

#### 3.2.2 字段配置
- **功能描述**：配置导入表的字段类型
- **自动识别**：
  - 整数 (INTEGER)
  - 小数 (FLOAT)
  - 文本 (VARCHAR)
  - 日期 (DATE)
  - 时间 (TIME)
  - 日期时间 (DATETIME)
  - 布尔值 (BOOLEAN)
- **手动调整**：
  - 用户可修改字段类型
  - 预览字段样本数据
- **表名设置**：
  - 默认使用文件名
  - 支持自定义表名

#### 3.2.3 数据导入
- **功能描述**：将配置好的数据导入 DuckDB
- **业务规则**：
  - 创建对应数据表
  - 批量插入数据
  - 导入完成后刷新表列表
  - 显示导入结果（行数、耗时）

---

### 3.3 数据查询模块

#### 3.3.1 表结构浏览
- **功能描述**：以树状结构展示数据表和字段
- **显示内容**：
  - 表名
  - 字段名
  - 字段类型
  - 行数统计
- **交互功能**：
  - 点击展开/折叠字段列表
  - 点击字段名插入到 SQL 编辑器
  - 搜索过滤表名和字段名

#### 3.3.2 SQL 编辑器
- **功能描述**：编写和执行 SQL 查询
- **编辑功能**：
  - 行号显示
  - Tab 缩进
  - 自动换行
- **快捷键**：
  - Ctrl+Enter：执行查询
  - Ctrl+Shift+F：格式化 SQL
- **安全限制**：
  - 禁止执行危险操作（DROP, DELETE, TRUNCATE 等）
  - 防止 SQL 注入

#### 3.3.3 查询结果
- **功能描述**：展示 SQL 查询结果
- **显示功能**：
  - 表格形式展示
  - 列名排序（点击列头）
  - 分页显示（每页 100 行）
  - NULL 值特殊显示
- **统计信息**：
  - 总行数
  - 查询耗时

---

### 3.4 数据导出模块

#### 3.4.1 Excel 导出
- **功能描述**：将查询结果导出为 Excel 文件
- **导出内容**：
  - 完整查询结果（不受分页限制）
  - 保留字段名
  - 保留数据类型
- **文件命名**：查询结果_日期时间.xlsx
- **触发方式**：点击"导出"按钮

---

### 3.5 数据可视化模块

#### 3.5.1 图表配置
- **功能描述**：配置数据可视化参数
- **配置项**：
  - 数据源：选择数据表
  - 图表类型：柱状图、折线图、饼图、环形图
  - X 轴/标签：选择分类字段
  - Y 轴/数值：选择数值字段
  - 聚合方式：求和、平均值、计数、最大值、最小值

#### 3.5.2 图表展示
- **功能描述**：生成并展示图表
- **交互功能**：
  - 鼠标悬停显示数值
  - 图例点击筛选
  - 响应式自适应

---

### 3.6 数据备份模块

#### 3.6.1 创建备份
- **功能描述**：备份当前数据库和上传文件
- **备份内容**：
  - DuckDB 数据库文件
  - 上传的原始文件
- **存储位置**：backups/ 目录
- **命名规则**：backup_日期时间

#### 3.6.2 备份列表
- **功能描述**：展示所有历史备份
- **显示信息**：
  - 备份名称
  - 创建时间
  - 备份大小

#### 3.6.3 恢复备份
- **功能描述**：从备份点恢复数据
- **业务规则**：
  - 需要用户确认
  - 恢复后刷新数据表列表
  - 当前数据将被覆盖

## 4. 非功能需求

### 4.1 性能需求
- 页面加载时间：< 2 秒
- 查询响应时间：< 1 秒（10 万行数据）
- 文件导入速度：> 1000 行/秒
- 支持并发用户数：10+

### 4.2 安全需求
- 密码加密存储（SHA-256）
- Token 认证机制
- SQL 注入防护
- 危险操作拦截
- 文件类型验证

### 4.3 可用性需求
- 响应式设计，支持 PC 端
- 直观的操作界面
- 清晰的错误提示
- 操作引导和帮助文档

### 4.4 兼容性需求
- 浏览器：Chrome 80+, Firefox 75+, Edge 80+
- 操作系统：Windows 10+, macOS 10.15+, Linux

## 7. 部署方案

### 7.1 环境要求
- Python 3.8+
- pip 包管理器
- 2GB+ 磁盘空间

### 7.2 部署步骤

1. **安装依赖**
   ```bash
   cd backend
   pip install -r requirements.txt
   ```

2. **启动服务**
   ```bash
   python app.py
   ```

3. **访问系统**
   - 浏览器打开 http://localhost:8000

### 7.3 生产环境部署

建议使用以下方式部署：

- **进程管理**：使用 systemd 或 supervisor
- **反向代理**：使用 Nginx
- **HTTPS**：配置 SSL 证书
- **日志**：配置日志轮转

## 9. 风险评估

| 风险 | 影响 | 应对措施 |
|------|------|----------|
| 大数据量性能问题 | 高 | DuckDB 优化 + 分页查询 |
| 文件格式兼容性 | 中 | 多编码支持 + 错误提示 |
| 并发访问冲突 | 中 | SQLite 锁机制 + 重试 |
| 数据安全风险 | 高 | 加密存储 + 权限控制 |
