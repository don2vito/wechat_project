## 四、交互设计

### 4.1 文件上传交互

| 触发条件 | 交互行为 |
|----------|----------|
| 鼠标悬停上传区域 | 边框变蓝色，背景变为浅蓝 |
| 拖拽文件进入上传区域 | 边框变蓝色加粗，背景变为浅蓝 |
| 拖拽文件离开上传区域 | 恢复默认样式 |
| 释放文件到上传区域 | 触发文件解析，显示加载状态 |
| 点击上传区域 | 打开系统文件选择器 |
| 文件解析成功 | 上传区域变为已上传状态，显示文件名和删除按钮 |
| 文件解析失败 | 显示错误提示（红色文字），上传区域恢复未上传状态 |
| 两份文件均上传成功 | 自动触发数据清洗和看板渲染，GSAP 入场动画 |
| 点击删除按钮 | 移除文件，恢复未上传状态，清空看板 |

### 4.2 筛选器交互

| 触发条件 | 交互行为 |
|----------|----------|
| 点击筛选器下拉框 | 展开下拉选项列表 |
| 选择选项 | 下拉框收起，区域2和区域3立即更新（带 GSAP 过渡动画） |
| 切换筛选器 | 数据重新聚合，表格淡入更新 |

### 4.3 表格交互

| 触发条件 | 交互行为 |
|----------|----------|
| 鼠标悬停表格行 | 行背景变为浅蓝 #EFF6FF |
| 表格数据超出最大高度 | 出现垂直滚动条，滚动条样式自定义（细滚动条） |

### 4.4 刷新按钮交互

| 触发条件 | 交互行为 |
|----------|----------|
| 鼠标悬停刷新图标 | 图标变为蓝色 |
| 点击刷新按钮 | 图标旋转动画，清空所有数据，恢复初始空状态 |

## 七、状态设计

### 7.1 空状态（初始状态）

```
+------------------------------------------------------------+
|  [Upload icon]                                             |
|  请上传当期和对比期 Excel 文件以开始分析                     |
|  支持 .xlsx 和 .xls 格式                                   |
+------------------------------------------------------------+
```

- 图表区域和表格区域显示空状态占位
- 筛选栏显示但处于禁用状态（灰色，不可点击）
- "大分类"下拉框仅有"全部"选项

### 7.2 加载状态

- 文件上传后，上传区域显示加载动画（旋转图标）
- 图表和表格区域显示骨架屏或加载动画
- 筛选栏保持禁用状态

### 7.3 已上传状态（单文件）

- 已上传的文件区域显示文件名和删除按钮
- 未上传的文件区域保持可上传状态
- 图表和表格区域保持空状态
- 筛选栏保持禁用状态

### 7.4 已上传状态（双文件，数据正常）

- 两个文件区域均显示已上传状态
- 图表区域渲染混合图表
- 表格区域渲染数据表格
- 筛选栏启用，"大分类"下拉框填充品类选项
- 所有区域带 GSAP 入场动画

### 7.5 无数据状态（筛选后无匹配）

- 图表区域保持不变（不受筛选影响）
- 品类分钟利润表格和播出档数表格显示"暂无数据"提示
- 筛选栏保持可用状态
